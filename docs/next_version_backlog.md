# 次版バックログ

## 目的

`Ver.02.00` 時点で見えている改善候補を、

- 次のバージョンで取り組む項目
- その次のバージョンで取り組む項目
- 機能面が固まった後に取り組む整理・最適化

に分けて整理する。

本ドキュメントは、コードレビュー結果、テストユーザーのフィードバック、
運用上の重要度を踏まえた計画の基準とする。

## 方針

- まずは **実験運用の信頼性と UX** を優先する
- 次に **実験効率を上げる機能追加** を行う
- 最後に **保守性・構造改善・最適化** を行う

## 完了済みの直近リリース

### `Ver.02.01` で完了した項目

### 優先度 A: 最優先で取り組む項目

#### 1. SerialWorker のブロッキング I/O 対策

- 背景
  - 現状は `readline()` の待機中に停止要求が入ると、切断時の応答が鈍くなる可能性がある
- 目的
  - GUI の切断応答性と安定性を上げる
- 実装の方向
  - 非ブロッキング寄りの読み取り方式へ変更
  - `stop()` から確実に抜けられる設計へ変更

#### 2. CSV / セッション耐障害性の強化

- 背景
  - 途中クラッシュ時にも、できるだけ最新のデータを救いたい
- 目的
  - 長時間実験中のデータ喪失を減らす
- 実装の方向
  - flush 制御の再設計
  - 必要に応じて一時ファイルや未完了セッション検出を導入

#### 3. Record 中であることの視認性向上

- 背景
  - ユーザーから「記録中がもう少し分かりやすいとよい」と要望あり
- 目的
  - 記録忘れ、記録中の誤操作を防ぐ
- 実装の方向
  - 赤系の枠強調
  - 蛍のようなふんわりした点滅演出

#### 4. Stop 時の確認ダイアログ

- 背景
  - 誤って記録終了しないようにしたい
- 目的
  - 実験中断事故の防止

#### 5. アプリ終了時の確認ダイアログ

- 背景
  - 誤って GUI を閉じないようにしたい
- 目的
  - 長時間実験中の事故防止

#### 6. 表示スパンボタンと手動パン・ズームの整合改善

- 背景
  - 手動で X 軸を操作した後に、同じ span を再選択しづらい
- 目的
  - グラフ操作の意図を UI へ正しく反映する
- 実装の方向
  - パン・ズーム時は span ボタンを全解除
  - 再度 span ボタンを押せるようにする

#### 7. セグメントのグラフ可視化

- 背景
  - ユーザーは「どこがどのセグメントか」をグラフ内で見たい
- 目的
  - 後から実験を解釈しやすくする
- 実装の方向
  - 下側グラフのみ
  - グラフ上部 10% 程度の帯表示
  - セグメントごとに背景帯を描画

#### 8. ヒータープロファイルの保存 / 再読込

- 背景
  - ユーザー定義プロファイルを名前付きで保存したい要望あり
- 目的
  - 実験条件の再利用性を上げる
- 実装の方向
  - 名前付きプリセット保存
  - 次回起動時に読み込み
  - プルダウン等で選択してロード
  - 起動時のデフォルト挙動は現状維持
- 実装状況
  - `codex/v02.01-profile-presets-and-packaging` で実装済み

#### 9. Windows のスリープ / 画面オフ抑止の保証強化

- 背景
  - 長時間実験中のスリープや画面オフを確実に避けたい
- 目的
  - 実験継続性を守る
- 実装の方向
  - 現行実装の確認強化
  - 必要なら Record 中だけでなく接続中の動作も再検討

#### 10. packaging 方針の見直し

- 背景
  - 起動速度重視で `onefile` より `onedir` を許容する方針
- 目的
  - ユーザー体験の改善
- 実装の方向
  - Windows 配布の主経路を `onedir` ベースで再評価
  - 起動速度と配布のしやすさのバランスを取る
- 実装状況
  - `PyInstaller onedir` に切り替え済み
  - Windows 実機でスモークテスト済み

#### 11. 起動中スプラッシュ / 起動状態表示

- 背景
  - ダブルクリック後、起動できているのか分かりにくい
- 目的
  - 起動待ち時の不安を減らす
- 実装の方向
  - 小さなスプラッシュまたはロードウィンドウ
  - ロゴと「起動中」表示
- 実装状況
  - `codex/v02.01-profile-presets-and-packaging` で実装済み
